## 一、系统概述

电梯报价系统用于根据客户选择的配置项，自动计算电梯整机报价。报价分为以下几个模块：

1. **基础配置报价** — 载重、速度、层站、曳引机、门机、控制系统、开门方式
2. **装潢报价** — 轿厢内部装潢、门装饰、吊顶、扶手、地坪等
3. **选配功能报价** — ARD、AVR、地震仪、IC卡、CCTV等

---

**基准价格：¥49,000**

## 五、报价计算公式（已确认）

### 基础配置报价公式

```
基础报价 = ¥49,000（基准）
载重加价 = (载重档位 - 1) × ¥2,000
速度加价 = (速度档位 - 1) × ¥2,000
层站加价 = (层数 - 10) × ¥2,000（最低层数3层，即最低-14,000）

基础配置报价 = 基准 + 载重加价 + 速度加价 + 层站加价
              （曳引机/控制器/门机/开门方式暂不走系统计价，走备注）
```

### 完整报价公式

```
整机报价 = 基础配置报价 + 装潢报价 + 选配功能报价

装潢报价 = 
  轿厢四壁材料加价之和 +
  轿底材质加价 +
  吊顶加价 +
  扶手加价 +
  轿门材料加价(×1) +
  厅门基站层材料加价(×1) +
  厅门其余层材料加价(×层数-1) +
  小门套基站层材料加价(×1) +
  小门套其余层材料加价(×层数-1) +
  显示板加价 +
  召唤盒加价

选配功能报价 = 各项选配功能加价之和
```

### 计算示例

**配置：630KG / 1.5m/s / 8层 / 普通装潢**

```
基准价格：       ¥49,000
载重加价：       (1-1) × ¥2,000 = ¥0      （630kg = 基准档位1）
速度加价：       (2-1) × ¥2,000 = ¥2,000  （1.5m/s = 档位2）
层站加价：       (8-10) × ¥2,000 = -¥4,000 （8层比10层少2层）
基础配置小计：   ¥47,000

装潢（假设全选普通钢板）：¥0
显示板（普通型）：¥0

整机报价：¥47,000
```
